# web-jnsports-notes

## 项目简介

本仓库用于归档和发布多个独立的 HTML 页面，内容涉及体育相关笔记与信息整理。仓库本身不针对任何特定网站或域名，所有页面均为独立静态 HTML 文件，可通过 GitHub Pages 等服务直接访问。

## 目录结构

```
web-jnsports-notes/
├── README.md          # 本文件
├── pages/             # 存放独立 HTML 页面
│   ├── page1.html
│   ├── page2.html
│   └── ...
└── assets/            # 页面引用的资源文件（图片、样式、脚本等）
    ├── css/
    ├── js/
    └── images/
```

- `pages/`：所有独立 HTML 页面均放置于此目录下，文件名与内容对应。
- `assets/`：页面所需的静态资源，按类型分目录存放。

## 页面归档说明

- 每个 HTML 页面均为完整独立的文档，不依赖外部服务或动态后端。
- 页面内容以体育笔记、资料整理为主，不包含商业推广或诱导性信息。
- 页面命名应简洁明了，便于直接通过 URL 访问。
